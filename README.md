

## Description du projet
Site web personnel de type CV en ligne, responsive et professionnel, réalisé dans le cadre du module **Optimisation Web** à l'École Supérieure de Technologie de Dakhla (Filière Marketing Digital & Expérience Utilisateur — 2ème année 2025/2026).

## Technologies utilisées
- **HTML5**
- **Bootstrap 5** (navbar, grille, cards, formulaire, boutons)
- **CSS personnalisé** (variables CSS, timeline, badges, animations hover)
- **Bootstrap Icons**
- **Google Fonts** (Raleway + Lora)
- **GitHub Pages** (hébergement gratuit)

## Sections du site
- Accueil / Profil (nom, titre, photo placeholder, bouton CV PDF)
- À propos
- Compétences (barres de progression + badges outils)
- Expériences & Formation (timeline)
- Projets (3 cards Bootstrap)
- Contact (formulaire + liens)
